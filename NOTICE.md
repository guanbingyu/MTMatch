# Notice

This public release bundles a cleaned MTMatch code snapshot, the TableMatch benchmark files, and compact result summaries.

The source code includes components derived from the Sudowoodo project and therefore retains the bundled BSD-3-Clause license. The TableMatch annotations were prepared for the MTMatch study. Upstream raw tables retain the terms of their original data sources.

Large transient artifacts, local logs, checkpoints, MLflow databases, manuscript build products, and unrelated baseline repositories are intentionally excluded.
